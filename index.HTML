The Rubber Duck Debugging Method:

  1. Obtain a rubber duck (bathtub variety).
  2. Place rubber duck on desk and inform it you are just going to explain some code.
  3. Go into detail and explain your code line by line.
  4. Tell the duck what you are doing next and then realise that that is not in fact what you are actually doing
  5. Fix code and be happy.
